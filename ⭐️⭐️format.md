# 店舗情報マークダウン生成プロンプト（最新版）

以下のフォーマットに従って、店舗情報のマークダウンファイルを生成してください：

## 基本情報入力

### 必須情報
- **店舗名**: [正式な店舗名]
- **住所**: [完全な住所]
- **サイトURL**: [公式サイトまたは食べログURL]

### タグ情報（カテゴリ別・色分け対応）

#### 1. 基本情報タグ（グレー）- 複数可
- **ジャンル**: イタリアン, 和食, 中華, フレンチ, カフェ, 居酒屋, ラーメン, etc.
- **特色・メニュー**: パスタ, 寿司, ラーメン, ステーキ, スイーツ, etc.
- **特徴**: オーガニック, スーパーフード, 地産地消, 手作り, etc.

#### 2. 価格タグ（グリーン）- 1つのみ
- **単一価格**: 1000円, 1500円, 2000円, 3000円, 4000円, 5000円, etc.
- **価格幅**: 3000円〜4000円, 2000円〜3500円, 1500円〜2500円, etc.

#### 3. 嬉しい情報タグ（ピンク）- 複数可
- **ファミリー**: 子連れOK, 子供メニュー, ベビーカーOK, キッズスペース
- **ペット**: ペット可, テラス席ペットOK, ドッグメニュー
- **シーン**: デート向け, 記念日向け, 女子会向け, 一人利用歓迎
- **対応**: 宴会可, 貸切可, 個室, 半個室, サプライズ対応, バリアフリー
- **特別サービス**: テイクアウト可, デリバリー, Wi-Fi, 駐車場

#### 4. アクセスタグ（青）- 1つのみ
- **駅からの距離**: [駅名][出口] 徒歩[X]分
- **例**: 大宮駅東口 徒歩4分, 新宿駅南口 徒歩2分, 渋谷駅ハチ公口 徒歩7分

#### 5. 営業情報タグ（黄色）- 1つのみ
- **営業時間・定休日**: [定休日情報 営業時間の概要]
- **例**: 不定休 11:00-24:00, 月休み 火-日11:30-22:00, 年中無休 24時間営業

### 画像情報
- **メイン画像URL**: [店舗のメイン画像URL]
- **ギャラリー画像URLs**: [複数の画像URL（オプション）]
- **画像レイアウト**: [表の行数×列数を指定]

### Description作成ルール
- **文字数**: 60-100文字程度
- **含めるべき内容**: 
  - 店舗の独自性・特徴（料理のこだわり、雰囲気の特色）
  - タグに含まれない情報のみ記載
- **含めてはいけない内容**:
  - 駅からの距離（青タグと重複）
  - 価格情報（グリーンタグと重複）
  - 子連れOK、ペット可などの対応情報（ピンクタグと重複）
  - 営業時間・定休日（黄色タグと重複）

## 画像配置指定方法

### 入力例
```
画像URLs: 
- https://example.com/image1.jpg (メイン)
- https://example.com/image2.jpg (ギャラリー)
- https://example.com/image3.jpg (ギャラリー)
- https://example.com/image4.jpg (ギャラリー)

レイアウト指定: 1行3列
```

### 自動判定ルール
1. **画像が2-3枚**: 1行で配置
2. **画像が4枚**: 2行2列または1行4列
3. **画像が5-6枚**: 3行2列または2行3列を推奨
4. **画像が7-8枚**: 4行2列または2行4列を推奨

## タグ作成の注意点

### DO（推奨）
- **価格タグ**: 幅がある場合は「3000円〜4000円」のように統合
- **アクセスタグ**: 「大宮駅東口 徒歩4分」のように詳細に
- **営業タグ**: 「不定休 11:00-24:00」のように要約
- **基本タグ**: ジャンル + 特色メニューの組み合わせ
- **嬉しい情報タグ**: 利用者にとってのメリットを重視

### DON'T（避ける）
- 価格タグを複数作成（3000円と4000円を分ける）
- アクセス情報をdescriptionに重複記載
- 営業時間を詳細にdescriptionに記載
- タグと同じ情報をdescriptionに含める

このプロンプトにより、統一されたタグシステムと効率的な情報表示が可能な店舗マークダウンファイルが生成できます。
